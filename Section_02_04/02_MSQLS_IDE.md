- A Solution may have 1 or more projects

1. Go to File in the left top corner, click New ‚û°Ô∏è Project.

2. Select **SQL Server Management Studio Solution** (`Blanck Solution`)

3. Give a name for your solution and a path, and click üÜó.

4. In **Solution Explorer** right click on Solution Add ‚û°Ô∏è New Project.

5. Select `SQL Server Script`.

6. Give a name to your project and click üÜó. The new project will be reflected in the **Solution Explorer** sidebar.

8. In **Queries** right click `New Query`

1. Erase the default connection in folder **Connections**.

2. Then right click in folder **Connections** and select new connection. In Server Name write `.\`, enter your password in Password input and click üÜó.
